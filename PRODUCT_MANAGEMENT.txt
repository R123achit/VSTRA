COMMANDS:
---------

Add/Update Products:
  cd scraper
  npm run import

Test Database Connection:
  npm run test-connection

Fix Database Issues:
  npm run fix-db


HOW IT WORKS:
-------------

1. Products are stored in: scraper/scripts/importProducts.js
2. Run: npm run import
3. Products are saved to MongoDB
4. Your shop displays them automatically


TO ADD MORE PRODUCTS:
---------------------

1. Open: scraper/scripts/importProducts.js

2. Add products to the array:
   {
     title: "Product Name",
     price: 499,
     originalPrice: 999,
     discount: 50,
     image: "https://images.unsplash.com/photo-XXXXX?w=800&q=80",
     images: ["https://images.unsplash.com/photo-XXXXX?w=800&q=80"],
     rating: 4.2,
     reviewCount: 1234,
     category: "men-tshirts",
     productId: "UNIQUE001",
     brand: "Brand Name",
     description: "Description",
     inStock: true,
     sizes: ["S", "M", "L", "XL"],
     colors: ["Black", "White"],
     source: "flipkart"
   }

3. Save file

4. Run: npm run import

5. Refresh shop page!
